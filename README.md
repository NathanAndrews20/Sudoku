Sudoku
=========================

Nathan Andrews

The Classic Game of Sudoku

*HTML, CSS, and React JS*

You can play sudoku @ https://nathanandrews20.github.io/Sudoku/

Notes
--------------
* Used the ```fetch api``` to receive sudoku boards from external website
* Used react hooks to manage game state and conditional stylings of the board
* Used CSS to create a ```modal box``` that appears when the user wants to the change the game difficulty
